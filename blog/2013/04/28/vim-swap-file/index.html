Vim Swap File

Apr 28th, 2013

I have started to learn how to use Vim recently. The first thing I need to learn is to use Vim to write my blog in Octopress. The one advatage of using Vim is that you fire up Vim inside Termial. Itmight make workflow easier. However. when I finished my article and deploy rake. Here error came.

The solution is to put `set noswapfile’ in .vimrc. Disable swapfile.
