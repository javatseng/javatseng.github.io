How to Install Wget in Mac OS

Sep 2nd, 2013

Wget is a very powerful CLI tool. However, Mac doesn’t have it by default. There are several ways to install. The easiest way is to use homebrew to install. Of course, homebrew has to be installed beforehand. All you need to do is type one command line:

$ brew install wget
