Using Pinboard in Command Line

Sep 7th, 2013 | Comments

I have been using Pinboard for a while. It is a very powerful bookmarking service with minimum interface. Now I can use CLI with pinboard. All I have to do is to install a gem and put a api token.

Install

gem install pinboard-cli

Place API

place it in ~/.pinboard-token
