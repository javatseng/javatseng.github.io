Could Not Start Spork Server for RSpec

Sep 8th, 2013

Try to install Spork and Guard in order to run Rspec more efficiently. After installing both of them, I typed Guard. Then, the message Could not start Spork server for RSpec came out in terminal. In the end, I found out that when I shut down my VPN, the server started successfully. I guess it took too long to run the code with VPN.
