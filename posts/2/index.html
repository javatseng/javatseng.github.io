How to Install Wget in Mac OS

Sep 2nd, 2013

Wget is a very powerful CLI tool. However, Mac doesn’t have it by default. There are several ways to install. The easiest way is to use homebrew to install. Of course, homebrew has to be installed beforehand. All you need to do is type one command line:

$ brew install wget

Home Key on Macbook Pro

Jun 17th, 2013

Mac app TaskPaper is my daily-use GTD app. I have digg the shortcut of TaskPaper. However, there were three shortcuts I could not figure out. They are home key and page up/down key. There are no suck keys on my Macbook pro keyboard. After posting my question on TaskPaper forum. someone answered my question:

Home key = command+fn+left arrow

page down/up=command+fn+down/up arrow

Octoeditor

May 14th, 2013

I have been using Octopress for a while. Since I started to learn Vim, I had been thinking how to integrate Vim with Octopress. My idea is to creat, write, and upload a blog all in Vim. In this way, I don’t need to switch between termainal and Vim. After searching for a proper plug-in, I foud a plug-in called Octoeditor. With this plug-in, you can do it all in Vim with sevaral shortcuts. Here is my shortcuts:

Crate a blog map <Leader>on :OctopressNew<CR> List all blogd map <Leader>ol :OctopressList<CR> Rake Generate map <Leader>og :OctopressGenerate<CR> Rake Deploy map <Leader>od :OctopressDeploy<CR>
