I have been using MailMate frequently. Personally, I would think it is a very geeky and one of the best wmail softwares in Mac. One of the strengths is, compared to other email clients, it is highly configurable. I have added one keybord binding to MailMate. The below is how I i did it:

  1. Create a folder in ~/Library/Application Support/MailMate/Resources/KeyBindings/
  2. Create a file called Special.plist
  3. Type file name Special in Custom Key Binding in Preferences.
  4. Put your keyboard binding in Special.plist. The below is my binding:

In my case, I use g+u to navigate to Uread folder. To make it work, you need to find mailbox UIUD. You can find it in ~/Library/Application Support/MailMate/Mailboxes.plist.

How to Install Wget in Mac OS

Sep 2nd, 2013

Wget is a very powerful CLI tool. However, Mac doesn’t have it by default. There are several ways to install. The easiest way is to use homebrew to install. Of course, homebrew has to be installed beforehand. All you need to do is type one command line:

$ brew install wget
