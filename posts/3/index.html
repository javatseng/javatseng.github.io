如何在terminal里出现full Path和host Name

Nov 8th, 2012

最近在看PeepCode里的meet the command的教程，把Unix的基础系统的学了一下，其中发现视频terminal里面prompt的前缀竟然是host name和full path，再看看自己prompt的前缀又长又没意义，于是网上摸索了一下，该改法如下：

  • 出现full path

  • 在finder找到~/.bash_profile这个file

  • 在档案里面加上export PS1='\u@\H:\w$'这行命令就可以了

  • 更改host name

  • 直接到 System Preference>Sharing>edit 修改local host name就可以了
